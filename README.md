# Super Quick MkDocs Demo

## To setup:
```
poetry install --with docs
```

### To Install Poetry:
```
# install pipx for isolated poetry env
python -m pip install --user pipx
# install poetry via pipx
pipx install poetry
# validate your install
poetry --version
# outputs: Poetry (version 1.7.1)
```


## To run:
```
mkdocs serve
```
